三星已于日前官宣，将于美国时间2月11日推出全新的S20系列手机，拥有较上代更大的屏幕、更强的配置、更高的镜头像素等。那么相比S10系列、三星Note10+，三星S20系列在机身尺寸上又会有什么变化呢？毕竟这会影响到手机的手感，

三星S20搭载6.2英寸屏幕，机身尺寸为151.7 x 69.1 x 9.1mm，而三星S10屏幕为6.1英寸，机身尺寸为149.9 x 70.4 x 7.8mm。相比之下，三星S20机身更长、更窄，但是会稍厚一些。

定位更高的三星S20+正面配备了6.7英寸屏幕，机身三围为161.9 x 73.7 x 8.9mm，比三星S10+同样也是机身更长更窄以及更厚。三星S10+屏幕为6.4英寸，机身的三围157.6 x 74.1 x 7.8mm。

与另外两款手机不同，今年新加入的三星S20 Ultra对比的是三星Note10+。前者屏幕尺寸为6.9英寸，机身三围为166.9 x 76 x 10.2mm，后者屏幕尺寸为6.8英寸，机身三围为162.3 x 77.2 x 7.9mm。值得注意的是，三星S20系列手机均采用20:9比例屏幕，一定程度上会让机身显得较为修长，但同时更方便用户握持。去年的三星S10系列便在握持感上给用户留下了相当深刻的印象，预计三星S20系列也将会延续这一优势。另外，从总体上看，三星S20系列全系为单打孔设计，而且孔径更小，颜值也相比其他三款旗舰更高，真机效果可期。
