在今年1月份，三星正式向自己的用户推送了基于Android 10的One UI 2.0。虽然升级后的系统无论颜值还是适用性都大大提升，但遗憾的是三星Good Lock未能同步更新，这也成了许多用户选择不升级的理由。而今日，根据三星官方通知，Good Lock 2020版将正式开放下载。三星用户终于可以在One UI 2.0上体验这一强大的自定义应用了。

Good Lock是由三星开发的系统增强框架，其目的就是为了弥补其在软件交互体验上的软肋。默认提供了自定义下拉菜单、自定义表盘、自定义多任务列表、增强多窗口功能、定制导航栏五种功能。通过下载还有更多的自定义工具可以使用。可以说，三星将国产手机系统内置的各种快捷工具全部统一放在了这个APP中，保持了设置菜单的纯净简洁的同时，十分方便管理。可以说，为了用户体验，三星也在一步步改善功能。此外，三星此次Good Lock 2020版更新并不只是对One UI 2.0进行了兼容，还进行了许多优化和调整，以下是官方更新日志。
