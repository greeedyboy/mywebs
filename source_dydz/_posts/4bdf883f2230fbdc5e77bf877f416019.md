春节期间，家家户户都在为餐桌上的大菜摩拳擦掌。鸡鸭牛羊，各种肉类都得配齐；鱼虾蟹贝，平日少见的海鲜也得尝尝。

“哎？这是什么螺？”这句话十有八九会出现在过年期间的餐桌上，那今天我们就“防问于未然”，来认识认识**我国最常见的十种食用螺**，让你在饭桌上对它们如数家珍，收获七大姑八大姨的赞许！**01**皱纹盘鲍 Haliotis discus hannai

皱纹盘鲍。图片：Jan Delsing / Wikimedia对，第一个出场的就是“叫鱼不是鱼”的鲍鱼。鲍鱼和其他海螺一样，属于**软体动物门腹足纲**。虽然扁平的外形常常让人忘记它海螺的身份，但仔细观察它壳顶部的螺旋便能发现，它其实就是**最后一圈螺层特别大的海螺**！名列“海八珍”之首的鲍鱼，是古代达官贵人才能享用的美食，但如今却已成为平民百姓餐桌上的家常菜。这主要是因为养殖行业的飞速发展，使得鲍鱼越来越便宜了。了解更多鲍鱼的知识，戳。

鲜鲍的价格目前比较亲民，爆炒蒜蓉蒸做汤炭火烤均可。图片：Manji / flickr**02**棒锥螺 Turritella bacillum

棒锥螺。图片：Zachi Evenor / flickr棒锥螺的螺塔极高，外形酷似**螺旋钉**，因此广东部分地区称之为“钉螺”。巧的是，这一俗名与血吸虫的唯一中间宿主——“钉螺”重名了。实际上，能传播血吸虫病的是生活在**淡水的钉螺**，生活在海水中的棒锥螺和它一点关系都没有，可以放心食用。棒锥螺最常见的做法是辣炒，一般会**先用钳子把螺的尾部夹断**，易入味的同时也更方便挑肉吃。

爆炒棒锥螺。图片：左撇子厨子 / 豆果美食**03**方斑东风螺 Babylonia areolata

方斑东风螺。图片：H. Zell / Wikimedia又名“花螺”、“凤螺”、“海猪螺”。东风螺腥味较轻、肉质鲜美、口感爽脆，在我国华南市场十分常见且畅销。一般的做法包括白灼、辣炒和酱爆。

香辣花螺。图片：慧慧小煮艺 / 豆果美食在众多海螺之中，东风螺算是十分“活泼”的类型。虽然平时潜伏在沙子里，但只要一嗅到食物的气味，就能**迅速钻出来觅食**，爬行速度让人怀疑它是不是打开了倍速播放。

快速奔向食物的东风螺。图片：berlika**04**荔枝螺 Thais sp.

某种荔枝螺。图片：me / Wikimedia又名辣螺、苦螺，有“螺中辣妹子”之称。荔枝螺生活在潮间带中下区的岩礁阴面或海底礁石上，在全国沿海均有分布，是**赶海最容易捡到的海螺之一**。它的口感偏辛辣，但不少人就好这一口，苦螺汤、百利炖螺肉等做法在闽南一带十分受居民和游客的喜爱。不过，在养殖鲍鱼的时候它的身份就只能掉为陪衬了——鲍鱼养殖户常会在鲍鱼网箱中放上几只荔枝螺，让荔枝螺这种肉食性螺类来**去除网箱中的牡蛎、藤壶、玻璃海鞘等生物**，从而提高鲍鱼的生长效率。荔枝螺：好好一只螺，怎么就混成保安了。**05**红皱岩螺 Rapana venosa

红皱岩螺。图片：George Chernilevsky / Wikimedia又名脉红螺、海螺，赶海视频里的“海蛎螺”一般也是指它。它在我国南北均有分布，且北方产量巨大，在市场上有稳定的供应。红皱岩螺足部肥大、口感清甜，颇受消费者的欢迎。

和荔枝螺一样，红皱岩螺也是**肉食性**的，它主食双壳贝类。遇到比自己小的贝类，它会先用腹足牢牢吸住，再分泌酸性物质将其麻醉，然后慢慢吃掉，因此部分地区还会用诱饵来“钓螺”。**06**大玉螺 Neverita didyma

大玉螺。图片：H. Zell / Wikimedia又名扁玉螺、香螺、肚脐螺。是我国沿海重要的经济螺类 , 其个体大、高蛋白、低脂肪、口感劲道、食后口余清香。大玉螺白灼、爆炒都很适合，不过烹煮时动作一定要快，否则**容易煮老**了，抠不出也嚼不动。

爆炒香螺。图片：赵涡涡 / 豆果美食大玉螺最大的特点，是它在舒展状态时显露出的肥厚肉足，在受刺激时又能迅速“挤水”变小，特别可爱，让人看了特有胃口。

大玉螺肥厚的肉足。图片：self / Wikimedia**07**泥螺 Bullacta exarata

泥螺。图片：尝尝好味道 / 豆果美食泥螺虽然个头不大，但是壳薄肉厚，一嘬一个吃起来非常过瘾。最常见的做法是“**醉泥螺**”：将泥螺清洗干净后，在盐水中多次浸泡，再放入茴香、桂皮、姜片等熬成的卤汁腌渍，最后加入少量黄酒，味道鲜甜口感脆爽的醉泥螺就完成了。醉泥螺是江浙地区的著名小吃，下饭指数五颗星。不过，由于**制作过程没有经过高温**，醉泥螺中可能含有**细菌或寄生虫**，建议煮熟后再食用。

熟醉泥螺。图片：鱼满仓海鲜 / 豆果美食**08**香螺 Neptunea arthritica cumingii

香螺。图片：Guido Poppe / gastropods又名响螺、风车螺、金丝螺。这种螺之前被称作**卡民氏峨螺**，在“香螺属”被划分出来之后它的中文学名就改为“香螺”了。

香螺个体一般在10厘米以上，螺肉既可鲜食，也可烘制成干，是一种经济价值很高的螺类。当然价格也不便宜，在辽宁省的市场价格近50元/斤。**09**椰子涡螺 Melo melo

椰子涡螺。图片：Mosmas / Wikimedia又名黄螺、肉螺、瓜螺。由于椰子涡螺生活在浅海容易采集，并且肉量可观，很早以前就是渔民重点采集的食用对象了。椰子涡螺可以长到哈密瓜的大小，重达20多斤，应该是你在海鲜市场上能见到的最大的螺。不过根据“吃螺第一定律”：**长得越大的螺一般肉质越老**。椰子涡螺也是如此：不熟则腥臭，过火又太硬，十分考验烹饪者的手艺。最佳的做法是把螺肉切成**一两毫米厚的薄片**，再与酸笋、辣椒等一起爆炒，让螺肉吸足调料的味道，又不至于嚼不动。**10**中国圆田螺 Cipangopaludina chinensis

中国圆田螺。图片：Wikimedia对于非沿海城市的人来说，以上几种海水螺并不多见，更常见的其实是生活在淡水的“田螺”。我们平时说的“田螺”，一般指的是**中国圆田螺**或**中华圆田螺**（Cipangopaludina cathayensis），这两种田螺在我国分布甚广，各地湖泊基本都能找到。

炒田螺。图片：云朵面包zx / 豆果美食吃田螺时最需要注意的是：千万别认错了呀！

田螺与大名鼎鼎的外来入侵物种**福寿螺**长得很像，而福寿螺是广州管圆线虫、卷棘口吸虫等寄生虫的中间宿主。一只福寿螺内可藏有6000条寄生虫，食用未充分加热的福寿螺可能会引发**寄生虫感染**，严重者可导致昏睡及死亡。咳咳，这哪儿是福寿螺呀，这是“折寿螺”吧。那么要如何分辨田螺和福寿螺呢？首先，福寿螺**个头比田螺大**，螺口也比田螺大；其次福寿螺的外壳呈**黄色**，而田螺呈青褐色；最后福寿螺的**“屁股”是扁平的**，而田螺是尖的。

左右滑动可看到福寿螺和常见的两种田螺，中国圆田螺和中华圆田螺的差别。图片：Claude & Amandine EVANNO上述这十种螺类，它们长相各异，口味也不同。但无论是哪种螺类，都有可能是寄生虫的中间宿主，只有完全煮熟之后才可放心食用。各位在新年大快朵颐的同时，一定要注意食用安全。牢记以下十二字箴言，让你安全过一个好年：**彻底熟食！彻底熟食！彻底熟食！**
