今天是世界湿地日。在湿地之上徘徊的群鸟里，**丹顶鹤**（Grus japonensis）是最美丽，最引人注目的鸟种之一。正如陆蠡在《鹤》所说，“鹤是长寿的动物，鹤是能唳的动物”，“鹤象征正直，鹤象征涓洁，鹤象征疏放”，自古以来鹤就受到中国人的喜爱。然而，丹顶鹤在中国却**命运多舛**。

丹顶鹤。图片：David Pike / NPL / Minden Pictures

**中国丹顶鹤的危机**在中国大陆上，丹顶鹤的种群可大致分为东部种群和西部种群。东部种群繁殖于黑龙江流域下游和三江平原等地，冬季迁移到朝鲜半岛越冬，数量约在1250只左右。西部种群几乎全生活在中国境内，在松嫩平原、黑龙江流域中上游和呼伦贝尔草原繁殖，在华东地区越冬，总数约400~500只。**西部种群的现状最令人担忧**。对丹顶鹤野外数量的评估显示，中国大陆的丹顶鹤种群呈现下降趋势。西部种群的迁飞区里面，“北大荒”曾经广袤的未开发土地，大部分已经被改造成为粮食主产区，而华东地区是开发程度最深，人口最为稠密的地区之一。**人类营造家园的活动**，却给丹顶鹤造成了沉重的生存压力。

“寒塘渡鹤影”。图片：Stephen Belcher / Minden Pictures丹顶鹤西部种群的数量仍在不断减少，与之形成鲜明对比的，却是中国**圈养丹顶鹤的数量在节节攀高**。2016年，中国以扎龙和盐城保护区为代表的五个保护区，圈养了518只丹顶鹤。到2019年，中国所有保护区和动物园的圈养丹顶鹤数量，已达2552只，**比所有野生丹顶鹤加起来还多**。在扎龙保护区，为了人工繁育丹顶鹤，工作人员甚至会**从野生丹顶鹤窝里偷蛋**，进行人工孵化。野生丹顶鹤受到惊扰可能会弃巢。即使不弃巢，虽然丹顶鹤会继续下蛋补充，但**孵化的时间已经耽误了**。小鹤出生晚，就不能在冬天到来之前及时长齐羽毛，和父母一起飞往越冬地。野外种群因此受害。（详情戳：）

日本北海道自然繁殖的丹顶鹤，亚成体的小鹤跟随着父母。图片：Masami Goto / Nature Production / Minden Pictures

**北海道丹顶鹤的隐忧**另一个位于**日本北海道**的野生丹顶鹤种群，可能会为我们目前的情况提供一些参考。丹顶鹤在日本曾广泛分布，但进入明治维新之后，由于栖息地被开发和狩猎，丹顶鹤的数量骤减。日本现存的丹顶鹤种群，是北海道的钏路（Kushiro）仅存的**20只左右丹顶鹤的后代**。1952年，严冬使溪流封冻，丹顶鹤面临饥荒，多亏当地人给这些鹤投喂食物，将它们拯救过来。在人们的帮助下，丹顶鹤的族群日益壮大。如今北海道丹顶鹤已**超过1600只**，而且数量还在增长。

北海道的保护区，很少结冻的溪流，为丹顶鹤提供了食物来源。图片：Claude Balcaen / Biosphoto / Minden Pictures丹顶鹤保护是日本野生动物保护的一大成功案例，然而成功背后也暗藏隐忧。过于集中的鹤群，容易爆发**传染病**。而且现在的北海道丹顶鹤，都源自20只左右的“祖先”，**遗传多样性相当低**，更进一步加大了疾病传播和出现遗传缺陷的风险。北海道的工作人员，会在冬季为丹顶鹤投食，而丹顶鹤的繁殖并没有受到干预。北海道丹顶鹤保留了大部分野生状态的行为，这不能不说是一件幸事。而扎龙保护区从野外取得丹顶鹤蛋之后，会进行人工孵化和饲养，也不回避**人与丹顶鹤的近距离接触**。保护区声称圈养丹顶鹤是为了“实现恢复壮大野生种群”，但大量的**圈养**丹顶鹤，是否能有助于这个**物种在自然中的延续**呢？

如今北海道的丹顶鹤种群数量已经达到一千多只。图片：Konrad Wothe / Minden Pictures

**来自美洲亲戚的提示**丹顶鹤的亲戚**美洲鹤**（G. americana），也是一种面临着种群缩减危机的大鸟，它或许能帮助我们理解，“笼中鸟”成功回归野生环境，并不是那么容易的。美洲鹤是目前世界上**数量最少的鹤类**（丹顶鹤是第二少的）。由于人们猎杀美洲鹤获取做饰物的羽毛，湿地被开垦为农田，以及电线造成的误伤，美洲鹤一度被逼到灭绝的边缘。1966年，美加两国的野生动物保护机构开始合作进行人工繁育美洲鹤。最初的尝试也是“偷蛋”，鹤每次通常会产两枚卵，但往往只会有一枚孵化存活。因此，工作人员会从中**取走一枚卵**，送到马里兰州的帕图森特野生动物研究中心孵化。随着**圈养种群**的建立，重新**向野外引入**美洲鹤的任务也提上了日程。

美洲鹤。图片：Jim Brandenburg / Minden Pictures1975至1988年间，总计有289枚源自野外和帕图森特中心的美洲鹤卵，被放入了爱达荷州格雷斯湖（Grays Lake）繁殖的野生沙丘鹤（G. canadensis）巢中。人们希望这些沙丘鹤能起到养父母的作用，抚养小美洲鹤长大，并且带着它们迁徙，以此来重建落基山脉一带的美洲鹤迁徙种群。但是，**从小跟着沙丘鹤养父母的美洲鹤，身份认同出现了问题**。长大成熟之后，它们**只会向沙丘鹤求偶**，对同类不感兴趣。1989年，计划被终止。该计划里的最后一只美洲鹤于2002年死去。吸取教训之后，人们又在佛罗里达州中部开始了新的尝试。1993至2006年间，289只幼鹤被放归野外。为了让它们从小学会避开人类，饲养员教人工孵化的幼鹤取食及饮水的时候，要**穿着模仿鹤的道具服**，**严禁在幼鹤面前说人话**，并且要播放鹤叫录音，**教会它们亲近同类而不是人类**。这一次，被释放到野外的美洲鹤生活得不错，但还是在繁殖上遇到了问题。这些仪态高雅的大鸟常常**莫名放弃正在孵化的卵或哺育的幼鹤**，从而导致孵化失败或雏鹤的死亡。

蒙住全身的工作人员用假鹤头指引小鹤觅食。图片：Mark Payne-Gill / NPL / Minden Pictures从2001年开始，另一个大胆的计划开始实施了。人们**用飞机“教育”美洲鹤学习迁徙**。最初，幼鹤被释放到威斯康辛州中部历史上的繁殖地，然后跟随超轻型飞机指引，**迁徙到佛罗里达州中部的越冬地**。截至2016年，总计有268只幼鹤被放归。这一次的尝试在重建迁徙行为上取得了成功，美国东部的天空又再一次出现了迁飞的美洲鹤。但是，人们还是没能教会美洲鹤怎么做父母。帕图森特中心的生态学家康弗斯（Sarah Converse）告诉我们，美洲鹤的繁殖行为，和它的**早期学习过程**相关，我们目前还**无法代替父母教育小鹤**，因此也不能让它们学会在野外正常繁殖。

飞机带领美洲鹤进行迁徙。图片：Tom Hugh-Jones / NPL / Minden Pictures到了2015年底，美国联邦野生动物管理部门决定做出改变，人们不再用飞机带领鹤群迁徙，也将逐渐减少人工喂养的幼鹤数量，今后工作的重点将放在“**更为自然地抚育幼鹤**和向野外释放个体”上面。由此可见，像养鸡养鸭那样拥有一大批圈养个体，并不能表示一种动物的保护活动取得成功，**野化的道路任重而道远**。虽然人类可以认识万物，但自然生物的复杂和多样，不是人类一时半会可以揣测透的。
