在人们的想象里，蝙蝠的形象往往与丑恶、恐怖联系在一起。除了布鲁斯·韦恩，形象最讨喜的蝙蝠恐怕要数白蝠了。白蝠属于叶口蝠科，生活在**中美洲**的加勒比低地（Caribbean lowland）地区。蝠如其名，它们浑身雪白，黄色的朝天鼻像迷你的猪鼻子，同时身材十分小巧，体不超过五厘米，体重五到六克，**仿佛一口一个的芒果糯米糍**。白蝠既不吃虫子，也不喝血，几乎只吃两种榕属植物 Ficus colubrinae 和 F. Schippii 的果实。

挤在一起的小可爱。图片：Phil Savoie / NPL / mindenpictures

**蝙蝠中的有房一族**白蝠昼伏夜出，白天栖息在芭蕉的亲戚蝎尾蕉属（Heliconia）的叶子底下。它们最特别的本领，就是把叶子稍加改造，做成栖身的“帐篷”。**已知会造“帐篷”的蝙蝠有22种**，大多属于叶口蝠科。在蝙蝠的进化史上，“造帐篷”这一技能被“发明”过多次，使用的**材料和建造技巧也是形形色色**。黄耳蝠属的 Vampyressa pusilla，会用喜林芋属（Philodendron）肥大的叶子造帐篷，它们**把叶梗和叶子相连的部分咬断一半，让叶子耷拉下来**，叶片底下就是它们的家。美洲果蝠属的 Artibeus jamaicensis 最爱圆形的棕榈科植物叶子，它们会在离叶梗不远的地方，**咬断一条条由叶梗延伸而出的叶脉，形成一个圈**，这样圈外的叶条就会一条条下垂，变成一圈遮挡的帘幕。简单的喜林芋帐篷一夜就能完成，而造棕榈帐篷的难度要大得多，造一顶棕榈帐篷要咬断六十多根又粗又硬的叶脉，需要好几天。

Vampyressa pusilla 这种蝙蝠同样以果子为食。图片：Phil Savoie / NPL / mindenpictures白蝠造帐篷时，会选择宽度超过十厘米、长度超过五十厘米的蝎尾蕉叶子。白蝠在叶片下方开始工作，它们**沿着中间主叶脉，****在离主叶脉大概半厘米的地方，咬出一个个小缝**，像给邮票打孔一样，然后用爪子扩大小缝的开口。叶片被咬破后，两边承受不住重量而下垂，就形成了一顶船形的帐篷。白蝠会保持叶子头尾两端完整，以支撑叶片的重量；它们还会在叶子中间咬一些小洞，可能是为了方便脚爪抓握。

**室友关系和睦**在绝大多数环境里，白色都非常显眼，容易被捕食者发觉，因此在雪地以外，白色的动物都比较少见。但白蝠不同，它们的白毛是**一种特殊的保护色**。白蝠呆在“帐篷”里，周围都是垂下来的绿叶，光线透过叶片变成绿色，白色的毛也被映成了绿色，跟叶片混为一体。

变绿的“糯米滋”。图片：Phil Savoie / NPL / mindenpictures研究发现，白蝠能在同一个地方生活很久，这说明它们不常遭到捕食，叶子有效地**遮挡了食肉动物的视线**。除了防备捕食者，白蝠的“帐篷”还可以**遮风挡雨**，减少热量散失。

展翅飞的样子还是很酷的。图片：Piotr Naskrecki / mindenpictures同一顶“帐篷”下居住的白蝠，数量从一只到十几只不等。“室友”关系相当亲密，天天住在一起，帐篷坏掉后，它们还会一起搬迁到另一片叶子下。白蝠**集体搬迁**的原因可能是，帐篷是一种比较珍贵的资源。蝎尾蕉叶子的使用寿命只有一个半月左右，时间长了“帐篷”就会破烂如杜甫的茅屋。所以白蝠必须定期找新叶子、造新帐篷。造新帐篷不仅要花力气，还会把显眼的雪白毛皮暴露在捕食者眼前。白蝠“室友”们一起找房搬家，可以**平摊风险**，只要一只白蝠找到了合适的叶子，大家都有房住。人手多，造帐篷的速度当然也会加快。

可能正在开卧谈会。图片：Leyo / wikimedia

**帐篷里的小“糕点”**科学家**曾经认为，白蝠的帐篷是一个“后宫”**。雄蝠制造帐篷，然后看守帐篷，把外来的雄蝠都赶走。雌蝠会被“有房人士”吸引，前来入住，成为房主的“妻妾”。这种“婚配制度”在进化生物学上称为“守护资源的多配偶制（Resourse Defense Polygyny）”。如果雌性动物需要某种东西（例如住房），而这种东西的分布又不均匀，集中在几小块地盘上，雄性只要把控住这种关键资源，就可以获得繁殖的权利。“帐篷是后宫”是一个漂亮的假说，但现实往往比理论更复杂。后来人们观察到，**雌蝠也会造帐篷**，不需要雄蝠帮助也能实现“住房自由”。而且，一片叶子底下常常有不止一只雄蝠，**雄蝠之间也不会为“争风吃醋”而打斗**。显然，雄蝠并没有表现出独霸“后宫”的能力和行为。对于白蝠“室友”的真实关系，我们还了解得很少。

“不要挑拨我们的关系哦。”图片：Tim Martin / NPL / mindenpictures刚生下来的白蝠宝宝就有满身的白绒毛，不过它们的鼻子不是橙色，而是粉红色。蝙蝠妈妈晚上出去觅食，把孩子留在帐篷里，随时回来给孩子★★、梳毛。有时母亲来不及回来，小蝙蝠会跟其他雌蝠要奶吃。我们还不知道，为什么白蝠乐意照顾别人家的孩子，也许是因为同住一个帐篷的雌蝠是亲戚，**照顾亲戚的孩子**，同样有助于自己基因的延续。小蝙蝠长得很快，出生第二天就可以睁开眼睛，二十天能飞，三十五天就长得和妈妈一样大了。

阿伟又死了是不是。图片：Phil Savoie / NPL / mindenpictures虽然叶子帐篷是很隐蔽的藏身之所，但如果你从它正下方抬头一望，还是能看到圆头圆脑、身披绿影的小蝙蝠们。它们看起来仿佛印尼的一种甜点——用班兰汁染成绿色，**沾满白色椰蓉的椰丝球**（Klepon），甜得人心都要化了。当然如果你在野外遇到它们，远观就好。养或者能好怎就别考虑了。

东南亚街头的椰丝球。图片：Onde Onde / flickr

当你凝望小可爱时，小可爱也在凝望着你。图片：Konrad Wothe / mindenpictures蝙蝠很厉害，但请放过蝙蝠自然界中的蝙蝠超过了一千种，它们是许多病原的天然宿主，其中不乏对人具有高致死率或高传染性的病毒，比如著名的狂犬病毒、尼帕病毒、马尔堡病毒、SARS病毒等。蝙蝠种类多，分布范围广，且善飞，同时频繁的人类活动日渐深入原始森林，这增加了病原在蝙蝠与其他动物和人之间传播的几率；而不少蝙蝠高度群居的特性，也使得蝙蝠之间变成了互相传播的“培养皿”。蝙蝠自己为何能够与这些病原相安无事，这是科学家们一直在攻克的问题。有人认为，飞行习性和高代谢率使得蝙蝠的体温高于其他一些哺乳动物，而高体温是病毒难以耐受的。另有一种解释，则与蝙蝠特殊的免疫系统有关。通常情况下，免疫系统会对侵入机体的病原产生响应，以抵抗或者消灭病原。这些响应的程度有所不同，有时程度较轻，免疫机制就已经清除了病原，此时你还没有生病的症状；有时程度较重，免疫系统与病原的战斗，以一些肉眼可见的形式呈现了出来，这就是发热、疼痛、流涕等症状。免疫系统的过度响应，即使能够杀死病原，也会对机体产生严重影响，甚至导致死亡。在人体，STING pathway 就是这样一种机制，但在蝙蝠体内，这种可引起强烈症状的机制被抑制了。

（doi:https://doi.org/10.1016/j.chom.2018.01.006）至于蝙蝠究竟如何与各种病原共存，目前尚待进一步的研究揭示。蝙蝠的唾液、粪便、尿液等均有可能带有病原，有传染给人的潜在风险。研究者接触蝙蝠尚有不同级别的防护措施，普通人敬而远之就好。本文是物种日历第6年第28篇文章，来自物种日历作者@红色皇后。
