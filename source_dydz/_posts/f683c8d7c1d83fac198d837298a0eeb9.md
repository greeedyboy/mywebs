在新一期的光环Wayoipoint更新日志中，343工作室公开了《光环：士官长合集》的更新计划，其中包括跨平台联机和自定义游戏等内容。

343表示，更新计划中的内容均计划在2020年内推出，但具体日期将取决于制作团队首要工作目标会如何变化，因此部分内容的推出日期可能会稍微推迟。

具体更新内容：1、跨平台联机（Crossplay）2、基于输入设备的多人匹配（Input Based MM）3、服务器地区选择（Sever Region Selection）4、自定义游戏Browser（Custom Game Browser）5、每款游戏的画面独立设置（Per Game Graphics Options）6、每款游戏的音效独立设置（Per Game Audio Options）7、Xbox键鼠支持（M&K Support for Xbox）8、PC文件共享（PC Fileshare）9、所有游戏的双键位设置（Double Keybinds for all games）10、所有游戏的视觉模型调整（Viewmodel adjustments for all games）11、游戏内帧数上限/调整（In game FPS Cap/Adjustments）12、Steam账户连接（Steam Account Linking）

343工作室称，第一项至第三项、第四项至第六项将同时更新，各位玩家敬请期待。
